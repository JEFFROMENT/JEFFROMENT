La page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui est envoyé ( adresse en hhtp://www.ect).C'est le navigateur qui interprète le code

Il existe 2 sortes de balises :

  1. les balises les plus courantes encadrent du code qui se termine par une balise dite fermante. Exemple :<balise>contenu HTML</balise>
  2. les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple : <br/> <hr/> <img/> <input/> ect...
